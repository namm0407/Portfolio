KKB Bus Stops

A Web app that allows users to locate nearby KMB (Kowloon Motor Bus) bus stops and the estimated time of arrival of bus routes at those nearby bus stops.

This Web app uses the Open Data provided by KMB through the HKSAR Transport Department. The Web app should be nicely rendered on any desktop computer and smartphone.
